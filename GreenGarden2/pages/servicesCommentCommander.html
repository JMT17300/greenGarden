Comment Commander

La première chose à faire est de vous rendre sur le site www.amazon.fr. Ensuite, suivez les étapes décrites tout au long de ce chapitre. Se connecter sur Amazon Comme annoncé au début de ce cours, le pré-requis pour commander sur Amazon est de posséder un compte. Si vous n’en disposez pas, suivez les instruction de mon cours pour créer un compte Amazon. Si vous n’êtes pas déjà identifié, cliquez sur le bouton Votre compte qui se trouve en bas à droite de la barre de recherche. Finissez en cliquant sur le bouton Identifiez-vous. Si vous n’avez pas de message d’erreur, vous êtes alors désormais connecté. Pour vous en assurer, vous devez voir apparaître votre nom à plusieurs endroit au niveau de l’entête du site. Remplir son panier Amazon Dès que vous êtes identifié, vous pouvez commencer à remplir votre panier d’achats, pour cela il suffit de : Faire une rechercher Consulter la fiche produit Ajouter l’article au panier La première étape consiste donc à rechercher ce que vous souhaitez acheter. Pour cela, plusieurs solutions s’offrent à vous soit vous naviguez dans les différentes catégories du site soit vous faîtes une recherche. Pour rechercher un article précis, le plus simple est d’utilise la barre de recherche qui se trouve en haut du site. Dans cette barre de recherche, vous pouvez saisir un mot clé, une catégorie ou directement une référence produit. Vous pouvez affiner vos recherche en choisissant une catégorie dans la liste déroulante qui se situe à gauche de la barre de recherche. Le résultat de la recherche est une liste de produits qui répond le mieux à vos critères de recherche. Pour trouver le meilleur produit, n’hésitez pas à utiliser les filtres pour afficher les produits avec 4 étoiles minimum, par prix… Consulter aussi les avis clients. Pour consulter le détail d’un produit, cliquez simplement sur son titre. La fiche produit contient, entre autre : * plusieurs photos de l’article, * le prix, * le détail des modes de livraison possibles, * une description détaillée * des avis et des questions/réponses de clients… Si vous souhaitez acheter le produit, cliquez sur le bouton Ajouter au panier qui est situé à droite de la fiche produit. Une fois que vous avez fini de remplir votre panier, l’étape suivante consiste à valider et payer la commande. La première chose à faire est de vérifier le contenu de votre panier en cliquant sur le panier tout en haut à droite sur le site. Si votre panier est correct, cliquez sur le bouton Passer la commande Adresse de livraison Amazon vous demande alors de renseigner une adresse de livraison. Pour cela, remplissez le formulaire en indiquant dans l’ordre: Nom et prénom Adresse postale Ville Code Postal Numéro de téléphone Finissez la saisie en cliquant sur le bouton Continuer. Si vous préférez choisir un point de collecte où récupérer votre colis, cliquez sur le lien « Trouver un point relais près de chez vous » qui se trouve en haut du formulaire. Amazon vérifie l’exactitude de votre adresse et est susceptible de vous proposer d’autres choix. Sélectionnez l’adresse qui correspond réellement à votre domicile. Mode de livraison Définissez ensuite le mode de livraison qui vous convient le mieux. Sachez que la livraison rapide, cochée par défaut, est la livraison classique dont le délai est 2 à 5 jours ouvrés. Elle est gratuite pour une commande supérieure à 25 euros. Les différents modes de livraisons sont exposés dans la suite de cet article. Valider le mode de livraison en cliquant sur le bouton Continuer. Coordonnées bancaires La page suivante vous demande de rentrer vos coordonnées bancaires afin d’effectuer le paiement de votre commande. Pour plus d’information concernant les possibilités de paiement, rendez-vous au chapitre comment payer sur Amazon. Une fois que les données de votre carte bancaire sont saisies, pour continuer votre achat cliquez sur le bouton Ajouter votre carte. Votre carte est alors enregistrée dans votre compte Amazon. Cliquez sur le bouton Continuer. Amazon est certainement l’un des sites les plus sécurisés au monde, donc vous n’avez aucun risque à laisser votre numéro de carte bancaire. Encart Amazon Prime C’est à cette étape qu’il faut être vigilant car Amazon met en avant une page de réclame pour sa carte de fidélité payante Amazon Prime. Vous pouvez la tester gratuitement pendant 30 jours. Par contre, si elle ne vous intéresse pas, vous devez cliquer sur le lien Continuer sans les avantages de livraison Amazon Prime.
